## How These Get Used

- **Website:** These words should appear naturally on quickstopsuperdeli.com — in page titles, headings, and descriptions
- **Google listing:** Posts on the Google listing should include these phrases
- **Social media:** Use location tags and hashtags that match these searches
- **Delivery apps:** Menu item names and descriptions should include relevant terms

## Tracking

Check monthly: search "deli Bellport" and "breakfast Bellport NY" on Google and see where we show up. If we're on page 1, it's working. If not, we need more reviews and more content with these keywords.
