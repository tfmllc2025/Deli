# /local-seo — Help People Find the Deli Online

Read `marketing/config/business-profile.json` and `marketing/config/brand-voice.md` first.

**What this does:** Helps more people find Quickstop Super Deli when they search Google for things like "deli near me" or "breakfast sandwiches Bellport." This is called local SEO — it just means making sure the deli shows up when people search online.

## Important: How to Write Your Output

The person using this is not a tech or marketing person. Follow these rules:
- **No jargon.** Say "Google listing" not "GBP." Say "shows up higher in search" not "improves rankings."
- **Step-by-step.** When something needs to be done, number the steps. Say which app/website to open, what to click, where to paste.
- **Copy-paste ready.** All text (posts, responses, descriptions) should be ready to paste with no changes needed.
- **One section at a time.** Don't overwhelm. If the user just says "run it," start with the most impactful item and ask if they want more.

## What to Do

Choose one or more of the following based on what the user asks (or do the most impactful one if they just say "run it"):

### 1. Google Listing Posts
Generate 2 posts to put on the deli's Google listing. Each should be:
- 150-300 words
- End with a clear action ("Call us," "Stop by," "Order delivery")
- Mention something seasonal, weather-related, or timely
- Include step-by-step instructions for how to post it (open Google Business Profile app → tap "Add update" → paste text → add photo → tap "Post")

### 2. Website Checkup
Look at quickstopsuperdeli.com and suggest simple improvements. For each suggestion:
- Explain what's wrong in plain English
- Explain why it matters (one sentence)
- Say exactly what to change (e.g., "Change the page title from X to Y")
- Flag anything that needs a web developer vs. what the owner can do himself

### 3. Google Review Response Templates
Write 3 ready-to-paste responses for Google reviews:
- **Good review** — Short, warm, mention something from their review
- **Bad review** — Stay calm, say sorry, offer to fix it, give the phone number
- **Okay review** — Thank them, address their concern, invite them back

Include step-by-step instructions: how to find the review on Google and where to type the response.

### 4. Business Directory Checklist
List websites where the deli should be listed (Yelp, Apple Maps, etc.). For each:
- Name of the site
- Is the deli already listed? (check if possible)
- Link to the signup/claim page
- Simple instructions to get listed

### 5. Website Code (Needs Developer Help)
Generate a block of code that helps Google understand the deli's info (called "structured data"). Flag this clearly as something that needs to be added to the website by someone who knows how — not the owner directly.
